Mortage Plan Application

Input activated commands:
1. "read doc" or "rdc" = reads the document in the folder and prints the result (basically the entire assignment)
2. "print all" or "pad" = prints all the customers found on the list
3. "get total" or "gtc" = returns and prints the number of registered customers
4. "exit" or "exi" = "closes down" the application

Maven and JUnit were used in this project
